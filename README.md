# CodeGenerator

### Getting Started

Add php-request to your application's composer.json.

```
{
  "repositories": [
    {
      "type": "vcs",
      "url": "https://github.com/fangqk1991/CodeGenerator"
    }
  ],
  ...
  ...
  "require": {
    "fang/code-generator": "dev-master"
  }
}

```

Run `composer install`
